# Bootcamp TOTVS — Dados, AWS & Machine Learning

Repositório com os projetos desenvolvidos durante o **Bootcamp TOTVS: Dados, AWS & Machine Learning**, uma trilha completa de 61 horas focada em engenharia de dados, computação em nuvem e aprendizado de máquina.
(spoiler: APRENDI MUITO em relação a computação em nuvem e machine learning, que eram meus focos principais ao realizar o bootcamp)

---

## Sobre o Bootcamp

O bootcamp oferece uma jornada que vai desde os fundamentos de Python e bancos de dados até Machine Learning, passando por ETL, cloud computing (AWS) e IA generativa.

### Estrutura

| Tipo | Quantidade |
|------|------------|
| Cursos em vídeo | 36 |
| Desafios de projeto | 3 |
| Desafios de código | 2 |
| Mentoria ao vivo | 1 |

---

## Conteúdo Programático

### 1. Princípios de Python e Versionamento de Código
- Fundamentos da linguagem Python
- Git e GitHub para controle de versão

### 2. Introdução à Ciência de Dados e Python
- Manipulação e análise de dados com Python
- Bibliotecas do ecossistema de dados (Pandas, NumPy)

### 3. Introdução a Banco de Dados Relacionais e ETL
- Modelagem de dados e SQL
- Processos de extração, transformação e carga (ETL)

### 4. Análise de Dados com Excel e Copilot
- Construção de dashboards e visualizações
- Uso de IA generativa (Copilot) para análise de dados

### 5. Computação em Nuvem (AWS)
- Como a computação em nuvem funciona
- Infraestrutura, computação e redes na AWS
- Armazenamento e banco de dados na AWS
- Estratégias de redução de custos com serviços AWS

### 6. Machine Learning
- Introdução ao Machine Learning
- Métodos bioinspirados e algoritmos genéticos
- Redes neurais artificiais
- Support Vector Machine (SVM)
- Classificação de problemas e exploração de datasets
- Linguagens de programação para ML
- Python para Machine Learning na prática

## Projetos Desenvolvidos

| # | Projeto | Descrição |
|---|---------|-----------|
| 1 | [Fluxo ETL com Python](codigo1/) | Pipeline ETL que consome uma API bancária, gera mensagens de marketing personalizadas com IA generativa (ChatGPT) e atualiza os dados via API REST |

## Benefícios e Aprendizados

- **Visão completa do pipeline de dados** — do tratamento e transformação até a geração de insights que orientam decisões de negócio
- **Experiência prática com cloud** — entendimento da infraestrutura AWS e como utilizá-la para reduzir custos operacionais
- **Fundamentos de Machine Learning** — base teórica e prática em algoritmos de aprendizado de máquina, desde redes neurais até SVM
- **IA Generativa aplicada** — integração de LLMs em pipelines de dados reais
- **Portfólio com projetos aplicados** — 3 projetos que simulam cenários reais do mercado, prontos para apresentar em processos seletivos
- **Mentorias com profissionais do mercado** — acesso direto a especialistas para tirar dúvidas e discutir aplicações práticas
